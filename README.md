# Ball Game
The genere of the game is a parkour game, and the idea of the game is the player is playing as shapes in order to complete each stage however for the most part of the game the player will be a sphere. At a certain part in the game the player will need to change its shape like cube, rectangle, and sphere for certain levels or stages in a level. The way to win the game is to get to the end of the level and get to the next stage, for the player to lose the player falls off the platform and it dies or rather the player goes back to its last checkpoint. As for UI plans we plan on adding the level on the top right and a menu system to be able to restart the level and possibly stage as well.
